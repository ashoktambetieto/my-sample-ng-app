# My Sample Angular App

## Overview

This is a sample Angular application built as part of a Proof of Concept (PoC) project using Backstage.

## Project Structure

```
my-sample-ng-app/
├── src/
├── docs/
└── package.json
```

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Angular CLI

### Installation

```bash
npm install
```

### Development Server

```bash
ng serve
```

Navigate to `http://localhost:4200/` to view the application.

### Build

```bash
ng build
```

The build artifacts will be stored in the `dist/` directory.

### Running Tests

```bash
# Unit tests
ng test

# End-to-end tests
ng e2e
```
